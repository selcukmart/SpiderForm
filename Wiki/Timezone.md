###### What Does This Class Do?

Presents the time zones of the countries in the form of a select box

###### How to use

```
                [
                'type' => 'timezone',
                'label' => 'Time Zone',
                'attributes' => [
                    'name' => 'timezone'
                ]
                 ]
```

Shows the time zones of all countries.

###### In Which Situations Is It Used

Used when a country's time zone is needed.
